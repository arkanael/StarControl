# Aplicação CRUD utilizando ASPNET MVC

Projeto desenvolvido em ASP.NET + Dapper

## Camadas:
### StarControl.Presentation

* Camada de apresentação, utilizando as demais estruturas;
* Cadastro simples de clientes;

### StarControl.Business
*Camada de regra de negócio

### StarControl.Infra.Data

* Repository Pattern: Repositórios Dapper para a manipulação de dados do banco de dados.

## Recursos Utilizados

* ASPNET
* C#
* Automapper
* Dapper
* Banco de Dados SQLSERVER
* JQuery
* Bootstrap
